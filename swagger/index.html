xTrain Event API (2.0.0)

Download OpenAPI specification:

A RESTful API to trigger events for the Unified Model Train Control System (xTrain). This version uses a dedicated endpoint for each event type. This API is based on the abstract interface defined in xDuinoRails_xTrainAPI.h.

Loco Speed Changed

Event for changing a locomotive's speed and direction.

Request Body schema: application/json
required
required
object (LocoHandle)

Unique identifier for a locomotive.

speedPercent
required
number <float> [ 0 .. 100 ]
direction
required
string (Direction)
Enum: "REVERSE" "FORWARD" "UNKNOWN"
speedSteps
required
integer

The speed step mode (e.g., 14, 28, 128).

Responses

Main API endpoint

/xTrainAPIv1/events/onLocoSpeedChanged

Request samples

  • Payload
Content type
application/json
Example
{
  • "loco": {
    • "address": 101,
    • "protocol": "DCC"
    },
  • "speedPercent": 100,
  • "direction": "FORWARD",
  • "speedSteps": 128
}
